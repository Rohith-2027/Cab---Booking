# Cab Booking Frontend

A React frontend for a cab booking system, integrated with a Node.js + Express + PostgreSQL backend.

## Features

- User authentication with JWT
- Role-based access (Customer, Vendor, Driver)
- Booking management
- Payment handling
- Notifications

## Tech Stack

- React (Vite)
- React Router
- Axios
- Context API
- Plain CSS

## Getting Started

1. Install dependencies: `npm install`
2. Start development server: `npm run dev`
3. Build for production: `npm run build`

## Backend Integration

Ensure the backend is running on `http://localhost:5000` with the specified API endpoints.
